Zoo Merge Adventure — Privacy Policy

Effective date: 3 July 2026

Zoo Merge Adventure ("the game") is developed by Joe Hardy and Matthew Stockdale ("we", "us").

The short version: we do not collect, store, or share any personal data. There are no accounts, no analytics, no advertising networks, and no tracking in the game.

Game data

Your zoo — progress, saves, and settings — is stored locally on your device. It is never uploaded to us or anyone else. Deleting the game deletes this data (unless your device backs it up as part of a normal device backup you control, such as iCloud).

Network use

The game makes one kind of network request: it periodically downloads a small configuration file (for example, event schedules and balance updates) from our static hosting. This is a download only — no personal data, identifiers, or gameplay data are sent with it beyond what any internet connection technically requires (such as an IP address, which we do not log or store).

Children

The game is suitable for all ages. Because we collect no personal data from anyone, we collect no personal data from children.

Changes

If a future update changes any of this (for example, if we add optional analytics or purchases), we will update this policy first and note it in the game's release notes.
